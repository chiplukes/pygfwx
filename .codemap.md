## Architecture Overview

```
pygfwx.encode() / pygfwx.decode()          ← Public API (codec.py)
        │
        ▼
encode_image() / decode_image()            ← Block pipeline (block_encoder/decoder.py)
  ├── write_header() / parse_header()      ← File format (header.py)
  ├── lift() / unlift()                    ← Wavelet (lifting.py)
  ├── quantize() / dequantize()            ← Lossy step (quantization.py)
  └── encode_coefficients()               ← Entropy coding loop
      / decode_coefficients()               (encoder.py / decoder.py)
          ├── get_context()               ← Neighbor statistics (context.py)
          ├── select_coding_mode()        ← Mode decision tree (context.py)
          └── interleaved_encode/decode() ← Golomb-Rice (golomb_rice.py)
              signed_encode/decode()        via BitWriter/BitReader (bitstream.py)

Supporting modules:
  transforms.py     — Color transforms (UYV/A710) — implemented, not yet wired
  bayer.py          — Bayer CFA sub-image handling
  multi_layer.py    — Multi-layer (stereo/depth) support
  metadata.py       — Application metadata chunks
  streaming/        — Progressive (truncated) decoding
  cross_codec/      — Reference SDK cross-validation
```
