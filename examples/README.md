# PyGFWX Examples

This directory contains example scripts demonstrating PyGFWX usage.

## Examples

### Basic Usage

- **basic_usage.py** - Simple encode/decode example demonstrating quality levels
- **cat_demo.py** - Generate compression metrics from test image

### Educational

- **compression_walkthrough.py** - Step-by-step walkthrough of the GFWX compression pipeline,
  explaining wavelet transforms, quantization, and Golomb-Rice coding with visualizations

### Advanced Features

- **progressive_demo.py** - Progressive decoding with truncation and downsampling

### Assets

The `assets/` directory contains:
- `cat_demo.png` - Test image for demos
- `cat_q*.png` - Compressed versions at different quality levels
- `progressive_*.png` - Progressive decode snapshots
- `downsampled_*.png` - Downsampled decode results

The `walkthrough_outputs/` directory contains visualization images generated by the
compression walkthrough script showing each stage of the compression pipeline.

## Running Examples

```bash
# Using uv (recommended)
uv run examples/basic_usage.py
uv run examples/progressive_demo.py
uv run examples/cat_demo.py
uv run examples/compression_walkthrough.py

# Or with Python directly (after installing)
python examples/basic_usage.py
```

## Requirements

The examples above require only the base `pygfwx` package (NumPy and Pillow).
No GFWX C++ SDK build is required.
